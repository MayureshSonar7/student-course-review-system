The Student Course Review System is a web-based application developed using Python Flask that allows students to view courses and submit reviews. The system provides a simple and user-friendly interface where users can register, log in, explore available courses, and share their feedback. This project is designed mainly for academic learning purposes and demonstrates the implementation of backend logic, database handling, and frontend integration using Flask.

This application follows a clean project structure with proper separation of concerns. Flask handles routing and backend logic, HTML templates are used for rendering dynamic pages, and CSS is used to enhance the user interface. The project also includes additional scripts for database testing and basic analysis, along with proper documentation.
